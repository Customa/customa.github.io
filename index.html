What is

?

Customa is a different Discord theme.
We offer the most customation that is possible, featuring both variables and modules,
creating a unique experience and giving the user full control over their client.
As of late we also feature support for White themes, compact mode works out of the box.

Who is

?

We are currently a team of 4, but you are free to join!
But Customa wouldn't be what it is today without all the people using it.
If the original Creator (GhostlyDilemma) wouldn't have gotten such great feedback over the course of almost 1.5 years,
the project would have been abandoned a long time ago.

Features

Customa Guarantee

The perfect base for a great Custom(a) Theme.
You get full support for every module and a lot of variables to tinker with.
If you want the full experience, here is where it starts.

Designed with BetterDiscord in mind

All settings for BetterDiscord have been themed according to the rest of the theme.
The theme works with and without BD Blue and Dark mode.

Deals with the Class-Mess so you don't have to

We've been dealing with all of the obfuscated classnames, in order to deliver a theme which runs perfectly on BetterDiscord, BeautifulDiscord, in the browser and on any other CSS injector.

12 variables and still counting!

12 custom variables are shipped with the base module alone, making the base itself a worthy alternative to the default CSS.
An explanation of the variables can be found later on.
